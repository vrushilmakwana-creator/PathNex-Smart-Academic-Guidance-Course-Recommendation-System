# PathNex-Smart-Academic-Guidance-Course-Recommendation-System
Smart academic guidance system built with PHP and MySQL that provides personalized course and college recommendations based on student interests, subjects, and academic background.

PathNex is a web-based academic guidance platform that helps students discover suitable courses and colleges based on their educational background, subjects, interests, and career goals. The system analyzes student profiles and generates personalized recommendations, making the process of selecting future academic pathways more informed and efficient.

Designed with a student-centric approach, PathNex combines profile management, course discovery, recommendation logic, and college exploration into a single platform. It also includes an administrative panel for managing courses, colleges, and user data.

Features
Personalized course recommendations based on academic background and interests
Student profile management
Course exploration and filtering
College recommendation and discovery
Admin dashboard for managing users, courses, and colleges
Secure authentication and session management
Responsive and user-friendly interface
Database-driven recommendation engine

How It Works

PathNex collects information such as a student's stream, subjects, interests, and preferred career direction. The recommendation engine evaluates this information against predefined course criteria and generates personalized suggestions. Students can then explore relevant courses and view colleges offering those programs.
